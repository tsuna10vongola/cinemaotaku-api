## Cinema Otaku API

This project is created to teach you how to create an API for an anime streaming website with Node.js, Express and MongoDB.

If you want to learn how the project is developed. You can visit https://www.youtube.com/watch?v=FPYlicctQMM&list=PLbKN8A2wssqUlVHRBeJIgIvkbyrX4kR0V or https://www.youtube.com/watch?v=9OfL9H6AmhQ&feature=youtu.be

### API Features

The application can create, read, update and delete data, for example: animes and episodes, in a database.
